## 🚧 Beta Release

**Rufus-Py** is currently in **Beta**.

It is a physical drive imaging and formatting utility written in Python, inspired by **Rufus**, with the goal of delivering a comparable experience for Linux users.

While core functionality is being implemented and refined, the project is still under active development. Users should expect bugs, incomplete features, and ongoing structural changes.

---

## 🎯 Aim

Rufus-Py aims to:

* Bring familiar USB imaging workflows to Linux
* Simplify bootable media creation for newer Linux users
* Provide a clean, minimal, and accessible interface
* Build a lightweight, Python-based foundation for future expansion

---

## ⚠️ Beta Disclaimer

* Not production-ready
* Features may change without notice
* Data-destructive operations should be used with caution
* Thorough testing across distributions is still ongoing

If you rely on stable, production-grade imaging tools, consider established alternatives until Rufus-Py reaches a stable release.
